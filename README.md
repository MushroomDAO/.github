# MushroomDAO

Organization configuration and community health files for [MushroomDAO](https://github.com/MushroomDAO).

## About MushroomDAO

MushroomDAO is building **Mycelium Protocol** — digital public goods infrastructure for community-driven governance and coordination.

|---------|-------------|
| [Park Protocol](https://github.com/MushroomDAO/Park-PublicGoodsGarden) | Public goods garden |

| [Sin90](https://github.com/MushroomDAO/Sin90) | OS for Individual |
| [CityOS](https://github.com/MushroomDAO/CityOS) | City operating system |
| [MyTask](https://github.com/MushroomDAO/MyTask) | Task marketplace |
